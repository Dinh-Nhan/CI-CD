Bookmark API (1.0.0)

Download OpenAPI specification:

Đinh Nhân: api@example.com

API quản lý bookmark (CRUD, search, filter, pagination).

Authentication

Tất cả endpoint đều yêu cầu Bearer JWT token. Để public một endpoint (ví dụ health check), override security: [] ở operation level.

Conventions

  • operationId theo pattern: {verb}{Noun} (ví dụ listBookmarks, createBookmark)
  • Lỗi server (5xx) luôn kèm trace_id để phục vụ debug
  • PATCH chỉ cập nhật các field được gửi lên; field không gửi sẽ giữ nguyên

Tạo bookmark mới

BearerAuth

required
url
required
string <uri> <= 2048 characters ^https?://

URL của bookmark. Chỉ chấp nhận scheme http hoặc https.

required
string [ 1 .. 255 ] characters

Tiêu đề hiển thị của bookmark

tags
Array of strings <= 50 items unique [ items [ 1 .. 50 ] characters ]

Danh sách tags. Nếu không gửi, server mặc định là []. So sánh unique theo case-insensitive.

string or null <uuid>

ID của folder chứa bookmark. Bỏ qua nếu không thuộc folder nào.

Cập nhật một phần bookmark

Cập nhật một hoặc nhiều trường của bookmark.

Lưu ý về tags: Nếu trường tags được gửi lên, toàn bộ danh sách tags hiện tại sẽ bị thay thế bởi giá trị mới. Gửi tags: [] để xóa hết tags. Không gửi trường tags nếu muốn giữ nguyên tags hiện tại.

Lưu ý về folder_id: Gửi folder_id: null để gỡ bookmark ra khỏi folder.

Authorizations:
BearerAuth
path Parameters
id
required
string <uuid>
Request Body schema: application/json
required
non-empty
url
string <uri> <= 2048 characters ^https?://

URL của bookmark. Chỉ chấp nhận scheme http hoặc https.

title
string [ 1 .. 255 ] characters

Tiêu đề hiển thị của bookmark

tags
Array of strings <= 50 items unique [ items [ 1 .. 50 ] characters ]

Thay thế toàn bộ tags hiện tại. Gửi [] để xóa hết tags. So sánh unique theo case-insensitive.

folder_id
string or null <uuid>

Gửi null để gỡ bookmark ra khỏi folder
